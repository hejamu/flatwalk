# flatwalk

Wang-Landau (flat-histogram random walk) sampling driver in Python.

Status: **M1 scaffold** — binning module implemented and tested. Core loop, checkpointing, and Ising validation arrive in M2/M3.

See the specification in `docs/spec.md` for the full design rationale, including the architectural choices that prepare for ≥2D order parameters and replica-exchange WL.
